Important changes:

  • Is now designed for Minecraft 1.19.4.
    • It's not advised to run this in older versions, I won't be able to provide much support if you do.
      (Imagine having to do that for all older versions for all my datapacks, you'll see a scale issue)

Gameplay changes:

  • Only players who used the Bone Meal onto the Composter gets the "Boosting Season" advancement.
    • Advancement progression is also reset for all players, and yes, the name has been changed.
  • Increased the chance for Glow Berries to appear to 12% (from 10%)
    • I've found that this feels more like vanilla after doing quite a few glow berry growth tests.
      (It's actually closer to 11%, but Composter adds more nutrients to plants, so 12% is fine)

Just me polishing some stuff in the background.Non-gameplay changes:

  • Codebase now runs on Scripter 3.1.
  • Optimized CropBooster's execution code.
  • CropBooster creation effect is now a bit more visible.
  • TheDiamondPlayables is now colored in dark aqua by design. (grey-ish aqua?)
    • The old design feels more vanilla, but it doesn't sit right with me.
      Seeing a gray TheDiamondPlayables just feel ... off.

Notes:

  • Experimental plants won't be supported by CropBooster, because this causes feature flag errors for those not playing in Experimental versions. You'll have to wait when it's officially released, I'll see what I can do when these plants are in official versions.
  • CropBoosters activated before this update still works normally after the update.
