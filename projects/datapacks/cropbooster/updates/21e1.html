Changes:

  • Is now designed for Minecraft 1.21.5!
  • CropBooster now supports Cactus Flowers!
    • They grow pretty much like how you'd expect them to in Vanilla, with a bit higher chance to grow.
    • Emulating this one accurately took me a while, but it works and runs pretty smoothly, so it's worth it.
  • CropBoosters no longer detach attached blocks when a boost level is lost.
    • This should be obvious behavior, but it can't be realistically done before 1.21.5.
  • Updated advancement background to match game update's general theme.

Admin Settings changes:

  • New Settings: Small/Medium/Tall Flower chance
    • Chance for a boosted Cactus to grow a Flower when it's 1/2/3 blocks tall respectively.
  • Moved Cactus's Growth Rate alongsides the Flower chance settings to its own Cactus Growth Settings page.
  • Reordered existing settings to keep the consistent look between pages.
  • Built-in presets have also been updated to support these new settings.
    • Presets made in previous builds will still work - it just assumes the new settings are unchanged.

Note: If the dates look off, it's because the official release went live that day without the IHF file improvements. I'm still counting it as it being released by then for simplicity sakes.
