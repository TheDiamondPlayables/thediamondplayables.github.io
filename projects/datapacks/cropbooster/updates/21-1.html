I really want to have more time to keep on doing things like this, but life stuff is pretty hard right now, which is what's really been dragging this back. You can help me get out of those problems (so I can get stuff out much quicker) over on Patreon - I really need your support there if you can afford it. Anyways, here are the changes:

CropBooster tweaks:

  • The CropBooster is now craftable by placing a Composter below a Bone Block!
    • This replaces the previous "throw a Bone Block in the Composter" setup method.
    • "Boosting Season" has also been reworked to account for this change.
  • Note that CropBoosters need to be placed in your line of sight to work properly.
    • There's a sound cue and some particles above the block to let you know when it works.
  • Breaking a CropBooster now drops it as an item, which lets you freely move it around!
    • A side effect of this: CropBooster might not work as well with packs that modifies the Composter's block drop. I'll tweak this if there is a better workaround.
  • The CropBooster block also has some tweaked functionality, but all these are side-effects and shouldn't be considered as mechanics.

Boosting tweaks:

  • CropBoosters no longer works on Sea Pickles.
    • Gameplay wise, setting it up is far too costly when compared to the benefits it brings.
    • Also because you really can't boost something that doesn't grow on its own.
    • As always, feel free to leave your opinions on any of my decisions on Discord!
  • CropBoosters now encourages maxed Melon/Pumpkin plants to deliver its crops!
  • CropBoosters' compost use has also been increased. Previous values are still too powerful.
  • Major improvements to project's boosting code performance!
  • Reworked boosted crops' growth rates to make it more reliant of the value it delivers.
  • Tweaked Glow Berry chance from 12% to 20% (about twice Vanilla's berry rates).
    • Makes the idea a bit more appealing if you're doing something like that.

Admin Settings:

Added /function tdp.config:cropboosterAdmin Settings to CropBooster, which lets you tweak things directly in-game to fit your worlds! Available options are:

  • Boost Range: Controls how wide the boosting square is.
  • Boost Rate: Percentage, controls how often CropBoosters boost crops!
  • Boost Loss Rate: Percentage, controls how often the CropBooster uses up Compost for boosts.
  • There is one for each supported plant!Boost Growth Rate: Percentage which controls how fast the crop grows when boosted.
  • Glow Berries' Fruit Rate: Percentage, controls how often a Glow Berry is made when boosted by CropBooster.

Extras:

  • Added Admin Commands! To use these commands:
    • Simply type /function tdp., and the game should give you suggestions on what to use.
      Side note: Any suggestions with underscores followed by numbers are for internal use only and won't work if run on their own.
    • Only those with the "Allow Commands" permission can use Admin Commands.
  • Admins can also give themselves custom items added by my projects with the new give Command!
    • Syntax: /function tdp.give:cropbooster/<item> {count:<1-6400>}
  • Fixed several inconsistencies between boosted Bamboo and regular Bamboo in specific cases:
    • Bamboo sizes not being large if boosted when tall enough.
    • Bamboo leaves duplicate in certain growth states when modified.
  • A lot of filenames have been changed to properly organize everything.
    • IDs have been removed and is now replaced by code names.
    • The format is now tdp:code_name/data, to reserve the old names for Admin Commands.
  • After 6 years of making datapacks, we finally have icons now!
  • Improved the look of info text messages.
  • Text files have been merged into one single Interactive Help File.
    • I have to learn 3 new languages to get all this to work as well as it did.
    • Not the best at it yet, but it works well enough for now.
  • The master background has been updated for major releases as always.
  • Codebase now uses Scripter 4.
