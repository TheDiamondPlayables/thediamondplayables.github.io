After a while of having to deal with landmines that came with Mounts of Mayhem, the Datapack Series are finally here for the last game drop of 2025!

Changes:

  • Is now designed for Minecraft 1.21.11!
  • Admin Books have been removed and replaced with the Admin Settings menu.
    • The Admin Settings menu lets you tweak all settings in one go, and apply them all at once!
    • Accessing this menu is the same as getting the Admin Book, so it should be easy to get used to it.
  • New Presets Menu:
    • This lets you view and edit a Custom Preset's values before applying them!
    • Triggers each time you load a Custom Preset.
  • New Show Configs Menu:
    • This lets all players read Admin Setting values in detail!
    • Access by running /trigger tdp.readconfigs.
    • Players will be notified on how to access this menu each time the Settings were changed.
  • The player who made changes to packs' Admin Commands are no longer announced in-game.
    • Mainly because that job is meant for the server consoles, and that information doesn't help with the players' gameplay experience anyway.
  • Tweaked the names, descriptions of all CropBooster's setting buttons to better match the interface it's now in.
  • IHF files now have their descriptions updated to reflect the recent set of changes.
  • Updated advancement background to match game update's theme.

Update notes:

  • Because the underlying system has to be reworked to support Admin Settings, your settings will be reset when you update to this version.
  • Don't worry though, you can bring your settings back instantly by loading a preset saved in the previous version, so be sure to do that before updating!

Dev comments:

  • If you're wondering why I'm dropping the Admin Books now: Mojang secretly obliterated a hidden mechanic that I used to make Admin Books feel nice to use, this was on top of the forced confirmation screen they added in 1.21.6.
    • Even if it's a bug that'll eventually be fixed, the damage has already been done, and I had to deal with it if I want to properly update all my packs to this release.
    • The reality is, adding new tools to make stuff with proper care just takes this long, and I have to make sure that the tool I'd built won't cause pack clashes when it's used on my other packs that were also running on Book Binder.
