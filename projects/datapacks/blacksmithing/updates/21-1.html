Changes:

  • Is now designed for Minecraft 1.21 and 1.21.1!
  • Changed how "Too Expensive" is removed:
    • Now you just need to click on the item while in the Anvil interface to remove it.
    • This is a simpler approach, that greatly improves the project's performance. Why not?
  • Anvil Recipes now supports the Crafter!
  • After 6 years of doing this (Blacksmithing existed since 1.13), we finally have icons now!
    • I'm not really good at drawing, so this is the best I've got for the project's icons.
    • Will probably improve it later on when I'm better at doing Art stuff.
  • Added Admin Commands! To use them:
    • Simply type /function tdp. in chat, the game should give you suggestions on what to use.
      • Side note: Any command suggestions with underscores are for internal use only and won't work if run on their own.
    • Only those with the "Allow Commands" permission can use Admin Commands.

Advancement Changes:

  • Due to the massive infastructure redesign, all advancement progress have been reset.
    • The infastructure redesign is aimed to keep things consistent with my other releases.
    • I considered keeping the Advancement files the same specifically, which saves your progress, but it also meant I'd have to manage 2 different filesystems internally, which messes up the workflow too much for it to ever be worth it.
  • Changed some advancement names:
    • Forging a split => Forging Fragments
    • Iron Forge Rebuilt => All Ironed Out
  • "All Ironed Out" now also accepts Anvil repairs from Crafters.

Extra Changes:

  • A lot of filenames have been changed to properly organize everything.
    • IDs have been removed and is now replaced by code names.
    • The format for commands is now tdp:code_name/data, to reserve the old names for Admin Commands.
  • Improved the look of info text messages.
  • Text files have been merged into one single Interactive Help File.
    • I have to learn 3 new languages to get all this to work as well as it did.
    • Not the best at it yet, but it works well enough for now.
  • The master background has been updated for major releases as always.
  • Codebase now uses Scripter 4.

I spend a lot of time keeping everything up to date, making things better each update.
