Changes:

  • Is now designed for Minecraft 1.21 and 1.21.1!
  • Concrete Powder can now be crafted using equal parts of Sand/Gravel, with the rest being other Concrete Powder!
    • This most likely won't be used often, but might be useful in some scenarios.
  • The original Concrete Powder redye recipe is now Any pattern would do, as long as you have the ingredients.Shapeless!
    • This is actually a part of the rule mentioned above, just that it has 0 extra Sand/Gravel mixed in.
    • Also because making Concrete Powder itself is shapeless, so it's a nod to that as well.
  • All Crafting Table recipes can now be automated using the Crafter!

Extras:

  • Added Admin Commands! To use them:
    • Simply type /function tdp. in chat, the game should give you suggestions on what to use.
      • Side note: Any command suggestions with underscores are for internal use only and won't work if run on their own.
    • Only those with the "Allow Commands" permission can use Admin Commands.
  • A lot of filenames have been changed to properly organize everything.
    • IDs have been removed and is now replaced by code names.
    • The format is now tdp:code_name/data, to reserve the old names for Admin Commands.
  • After 6 years of doing this (Colored Dyeing existed since 1.13), we finally have icons now!
  • Improved the look of info text messages.
  • Text files have been merged into one single Interactive Help File.
    • I have to learn 3 new languages to get all this to work as well as it did.
    • Not the best at it yet, but it works well enough for now.
  • The master background has been updated for major releases as always.
  • Codebase now uses Scripter 4.
