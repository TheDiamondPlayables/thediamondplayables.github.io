Minespenser lets Dispensers mine blocks using tools!
With a tool inside, Dispensers can be used to mine blocks!

Some notes with this:

  • This supports enchanted tools!
  • Blocks only drop items when the right tool is used.
  • Harder blocks takes more shots to mine through.
  • Mining with this is faster, though it costs a bit more durability.
I made this because I got too lazy mining smooth stone for a build.
