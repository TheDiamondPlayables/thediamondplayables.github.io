(Also known as "A New Seat" update)

Note: Starting a small experiment to try and categorize the new stuff in each update log.

Note if you're updating from an older version:

  • Be sure to uninstall the previous version before moving up to v19d1.
  • The code has been reworked to include all the new stuff from 1.19.4, and has also been optimized to run even better, too!
    • That also means that old stuff from previous versions won't run in v19d1, so if you want to keep things clean, uninstall v19c1 before updating!

Important Changes:

  • Is now designed for Minecraft 1.19.4 and higher.
    • It's not advised to run this in older versions, I won't be able to provide much support if you do.
      (Imagine having to do that for all older versions for all my datapacks, you'll see a scale issue)
  • You can now sit underwater!
  • You can now sit on Fences, Walls and Fence Gates!
    • You must be able to stand on top of the Fence in order to sit on it.
    • Fence Gates must be closed in order to be sat on.
      (which is kinda obvious, but still mentioning it just in case!)
  • For lower stair blocks: The sitting position is now always on the lower half of the block.
  • Sitting on top of Chests now locks it up while you're sitting on it.
  • The codebase was also redesigned for 1.19.4 to include some new features as well as making them run better overall.
    • The new approach came at a cost though: The cushion's been removed.
      I'll find a way to bring it back in the future, but for now it is what it is.
  • Fixed a case where you'd get previously "Time to Rest!""Time to Sit Down!" advancement without having to sit in a seat.
    • Because of this, "Time to Sit Down!" completion has been reset for everyone.
    • The advancement's EXP reward was also shifted to 15, from 10.
  • Removed more items from the valid items to sit in list.
    • Some items removed: Boats, Minecarts, End Crystal, Cocoa Beans, Sweet/Glow Berries, ...
    • These items are selected because you might sit down when you don't want to, but did because you absentmindedly crouched at the time.
    • Another note: You can still use everything when you've sat down.
