Who woulda thought a pack that lets you sit anywhere be this complex?
It doesn't have to be, sure, but that just means there's room for improvement, and improving it just takes you to where we are now. Anyways, here are the changes:

Changes:

  • Is now designed for Minecraft 1.21-1.21.1!
  • Improved how sitting works:
    • In order to sit, simply crouch twice in a short time to start sitting!
    • You can see why I changed it again - distinct, easy to do and doesn't require clearing your hands.
    • As always, I'll improve the methods whenever I can think of a better way, and this one seems promising so far ...
  • Upgraded seat placement position, letting you sit in a lot more places!
    • I've added 8 new spots per block, which'll be placed when you stand close to the edge of blocks.
    • Placement is center-focused, so if you're closer to the block's center, that's where the seat is.
      If you're closer to the edge of blocks, cushions will be placed there instead!
  • You can now move cushions around by hitting your Use key on them while crouched!
    • Where cushions are repositioned to depends on where you're looking at.
    • They'll silently fail if you wanted to place a seat in a bad spot.
    • These options let you set where you want to sit for the more complex set of block positions.
  • New sitting effects:
    • New sounds for sitting, seat placement, and standing up!
    • New particles when a seat is removed!
    • I can't believe I missed these little details, but it adds a lot more life to the pack.
  • Added Admin Settings, which lets you tweak some aspects of the new mechanics:
    To tweak these, run /function tdp.config:blockseat
    • Crouch Delay: Sets how long between crouches to register a double crouch.
      • Counts in 1/20 secondsticks. Defaults to 0.5 seconds10.
      • Do note that the game requires at least a 1-2 tick gap to register an uncrouch, so values too low can be hard to use.
    • Chest Blocked: Chests can be locked by sitting on them.
      • True/False. True by default.
    • Seat Effects: Adds sounds and particles to BlockSeat's cushions.
      • True/False. True by default. Turn off if you like the quieter feel.
  • "Time to Sit Down" advancement has been reset to teach players how to sit again.
    • The description has also been tweaked to reflect the new sitting method.
  • New advancement: "Cushion Mover"
    • Move a seat by pressing your Use key on a seat while crouched.
  • Fixed a bug causing Chests to be locked permanently if someone logged out while sitting on it.
    • Existing cases won't be fixed mainly because you can just replace the affected blocks yourself.
      Or choose not to and keep a piece of history with you. I won't judge.
  • Overhauled BlockSeat's internal execution to support the new mechanics.
    • This means there'll be a lot of weird little differences to watch out for.
    • Most of these won't affect the majority of players, but I thought you should know about it.
  • Reduced BlockSeat's reliance on fixed block sets.
    • This should make the pack work better with blocks added via mods, but no guarantees!

Extras:

  • Added Admin Commands! To use these commands:
    • Simply type /function tdp., and the game should give you suggestions on what to use.
      Side note: Any suggestions with underscores followed by numbers are for internal use only and won't work if run on their own.
    • Only those with the "Allow Commands" permission can use Admin Commands.
  • A lot of filenames have been changed to properly organize everything.
    • IDs have been removed and is now replaced by code names.
    • The format is now tdp:code_name/data, to reserve the old names for Admin Commands.
  • After 6 years of making datapacks, we finally have icons now!
  • Improved the look of info text messages.
  • Text files have been merged into one single Interactive Help File.
    • I have to learn 3 new languages to get all this to work as well as it did.
    • Not the best at it yet, but it works well enough for now.
  • The master background has been updated for major releases as always.
  • Codebase now uses Scripter 4.

Dev Notes:

  • I tried to support different seats based on player scales, but it wasn't worth adding.
    • No matter what I did, it always went past the performance budget, especially the giant seats.
    • Also because the bigger something is, the more likely it is to be bugged out when combined with the game's other technical mechanics like chunk loading.
