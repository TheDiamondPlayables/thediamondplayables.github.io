(Also known as the Strange New Names update.)

Changes:

  • Is now designed for Minecraft 1.20.2!
  • New Name Tag command: /rename
    • Villagers when applied with this Name Tag sets a new name for themselves.
    • You can get Special Names (colored names) this way for worlds with limited Villager spawns.
  • New Name Tag command: /nameless
    • Villagers when applied with this Name Tag are now nameless again.
    • Baby Villagers can also inherit this, so you can build a nameless village with it.
    • /realname can be used to remove the Villager's nameless effect.
  • New Advancements:
    • Nomen Renovatum: Change a Villager's Name with a Name Tag called "/rename"!
    • Reveal Yourself: Reveal a Villager's true name with a Name Tag called "/realname"!
    • Nameless Order: Hide a Villager's Name with a "/nameless" Name Tag!
    • That Villager is a Spy!: Trade with Villager Names's creator. Wait ...
  • Removed old "What's Your Name?" advancement in favor of the new "A Name Revealed".
  • Effect sounds caused by players using Name Tags on Villagers are now in the Players channel.
  • Villagers affected with Villager Names are now tagged with tdp.datapacks.
    • Pack makers could use this to exclude tagged entities I'd made to avoid combo issues, if there's any.
