(Also known as the Make or Break update)

Changes:

  • Now officially designed to work for Minecraft 1.19x!
  • Blocks replaced if you place a Ladder on it are now also replaced if placed with DropLadder.
    • Some examples: Grass, Vines, Sculk Veins, ... (21 more)
  • Ladders placed by DropLadder now causes vibrations, with a value of 13.
    • I know placing blocks gives a redstone value of 12.
      Making it match that is just way more trouble than it's worth.
  • Added Chainbreaking:
    • Ladder blocks when broken now breaks nearby ladders if they're not properly supported.
    • Chainbreaking will not work if Ladders can't drop anything, or drops something else.
      (I used block drop detection to make this work - lightest option performance wise)
  • Fixed a bug where Ladders would place itself when broken underneath another Ladder block.
  • DropLadder will now ignore Ladder items that have touched the ground.
    • Should improve performance in cases where there's many ladder items on the ground.
  • Updated advancements background and colors of most text found in the pack.
  • Codebase now uses Scripter 3.
