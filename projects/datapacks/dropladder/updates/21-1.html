Changes:

  • Is now designed for 1.21 and 1.21.1!
  • Added the Rocket Ladder that lets you drop Ladders upwards!
    • To craft Rocket Ladders, surround Ladders with a piece of Firework Rocket to make 8 of them!
    • Throwing Rocket Ladders at a Ladder sends it up, placing Ladders as it goes!
    • All Firework Rockets produce the same Rocket Ladder currently, so use the cheapest one you got.
  • Admins can also give themselves custom items added by my projects with the new give Command!
    • Syntax: /function tdp.give:dropladder/<item> {count:<1-6400>}
  • Added custom stacksizes support for Ladders and Rocket Ladders!
  • New Achievements:
    • Discount Rocket Science: Craft some Rocket Ladders with the cheapest Rockets you can find!
    • To Infinity ... And Beyond!: Send Rocket Ladders flying by throwing them onto a Ladder!
  • Tweaked description for "Let Down Your Ladder" to increase clarity in the wording.
  • Massively improved Chainbreaking performance, especially at scale!
    • The pack takes a while longer than my others to be installed in some cases, but it's well worth it afterwards.
  • Chainbreaking now seamlessly works with other custom made blocks added via mods.
    • In theory: If the mod respects vanilla Ladder mechanics, Chainbreaking should have no problems with it.
    • Note: This is nothing more than just a side-effect of the performance improvement change.
  • Fixed bugs:
    • Fixed an issue that could cause Chainbreaking to randomly fail if the Ladder is too long.
    • Fixed some edge cases that caused Chainbreaking to trigger where it's not supposed to.

Extras:

  • Added Admin Commands! To use these commands:
    • Simply type /function tdp., and the game should give you suggestions on what to use.
      Side note: Any suggestions with underscores followed by numbers are for internal use only and won't work if run on their own.
    • Only those with the "Allow Commands" permission can use Admin Commands.
  • A lot of filenames have been changed to properly organize everything.
    • IDs have been removed and is now replaced by code names.
    • The format is now tdp:code_name/data, to reserve the old names for Admin Commands.
  • After 6 years of making datapacks, we finally have icons now!
  • Improved the look of info text messages.
  • Text files have been merged into one single Interactive Help File.
    • I have to learn 3 new languages to get all this to work as well as it did.
    • Not the best at it yet, but it works well enough for now.
  • The master background has been updated for major releases as always.
  • Codebase now uses Scripter 4.
