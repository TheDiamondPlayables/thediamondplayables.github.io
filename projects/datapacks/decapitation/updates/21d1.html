Changes:

  • Is now designed for Minecraft 1.21.4! (Ported from Version 21b1).
  • Changed background and icon to match current game drop's theme.
  • Added Creaking Head, which you get by breaking a Creaking Heart while its Creaking is active.
    • Fortune is used instead of Looting in order to boost the head's drop rate.
    • All Creaking sounds can be found while brushing it.
  • Added Creaking imitation sound to Parrot Heads.
  • Added new Settings controlling how Creaking Heads drop in special scenarios.
    • Useful if you have a mod/datapack that controls specific mob griefing methods separately.

Creaking Head Details:

  • Creaking Head's drop sources are applied If unlinked Creakings are introduced via other packs/mods, the drops will work as long as it was also not modified by said packs/mods.to the mob itself and also to it's linked Creaking Heart.
    • For example: If a Charged Creeper blew up the linked Creaking Heart, the Creaking Head's source will be Charged Creeper.
  • For drop sources that can't deal block damage: If they are the only ones active, Player Kill conditions will apply instead.
    • Affected sources: Skeleton Kill, Potion Damage, Firework Explosion.
    • Affected depending on the Can be edited using the Admin Book.Mob Griefing Data: Ghast Fireball, Charged Creeper Blast, Creeper Blast, Wither Kill.
    • I had different approaches to dealing with this but in the end I decided that in general, if the head can't be obtained via the sources selected, it'll default to Player Kill to make it easy to wrap your head around, which is important if you're using a lot of mods/packs together.
