Changes:

  • Is now designed for Minecraft 1.21.5! (Ported from Version 21d1.)
  • Added Cold and Warm variants for Cow, Chicken and Pig Heads!
    • These heads have the same sound set as the default Heads.
    • Also, no, the classic heads won't have the Temperate prefix.
  • Updated the following heads to better match the game's updated models:
    • Cow Head
    • Red Mooshroom Head, Brown Mooshroom Head
    • Pig Head
    • Chicken Head
    • Because these are really close to the original textures, I've opted not to have an Old variant for them.
  • Added all the new wolf sounds to Wolf Heads!
  • Wolf Head Sound tweaks:
    • All wolf heads now share the Shake, Hurt and Death sounds.
      • Shake is more likely to show up for Dog Heads and less likely for Angry Heads.
    • Other sounds used by the head variants:
      • Regular: Woof
      • Dog: Previously called Sad, but we got a Sad wolf sound variant nowWhine, Pant
      • Angry: Growl
      • These sounds are more likely to show up compared to the shared sounds.
    • All vanilla sound variants have been added.
    • The "Howl" sound has been removed because it was removed by the game.
    • The new changes only apply when you're modifying the sounds.
      • Your existing mob head sound contraptions still works as they did, unless the block was mined, got brushed, or you used the Howl sound.
  • Reworked Tropical Fish head variant detection algorithm using new update's mechanics.
    • These head variants are now accurate even if the fish died instantly after spawning in.
  • Player Heads now stack together if their Cause of Death looks similar enough.
    • These items are now also stored more efficiently, which'll help if you have a lot of these.
  • Changed background and icon to match current game drop's theme.

Update Notes:

  • If you're using Decapitation from Branch 21-x to 21dx before this update, uninstall said branch first before updating!
    • This'll remove the extra Tropical Fish storage data that's no longer used in this version.
    • If you have modified settings: Save it in a Preset first, so you can load it again after the update!
