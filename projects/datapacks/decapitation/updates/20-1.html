(Also known as the "Heads of the Past" mega-update)

The "Heads of the Past" Mega Update is finally out now, with a huge overhaul towards how mobheads work to better compliment the new Mob Head Sounds mechanic introduced in Minecraft 1.20, new head variants, reworked head drop rates, and more tools to find how many heads you've found so far!

Important Note:

  • If you have Merchant or Fishing Extended, you'll need to update those 2 as well to avoid Pack Clashing.

Additions:

  • Mobheads now play sounds when used on a Note Block!
  • Brushing a new mobhead now changes the sound that mob plays above a Noteblock!
    • Note that each head variant has its own set of possible head sounds.
    • The head itself should give you a hint on which head sounds it has.
    • Due to game limitations, this doesn't work on mobheads made before v20-1, or vanilla mobheads.
  • Move a head with its sound by breaking it with the Brush!
    • You can also inspect the sound the head makes this way, too!
  • Mobheads now have extra attributes to show where you've got it.
    • This mainly serves to separate Heads collected between different methods.
    • Mainly because some methods are harder to collect than others.
  • Added new heads:
    • Screaming Goat Head.
    • A "Noisy" variant to Zombie Head and Skeleton Skull.
    • Camel Head.
    • Sniffer Head.
  • Hold Ctrl, then press your Pick Block keyCtrl-Picking now tries harder to emulate the old mob head's name.
    • This is a hardcoded game action though - I can only emulate it as hard as I can.
    • You can still break the mobhead by any indirect means in Creative to get the original head back.
  • New Advancements:
    • Noteworthy Dustoff: Brush a head to change it's sound!
    • What's that sound?: Break a brushed head with a Brush to find out!
  • Added a new Available Heads Excel file!
    • The file is made with Excel, so you'd need Excel or something compatible to open it.
    • Here's hoping your spreadsheet viewer/editor supports Filters, cause you're gonna need em.

Changes:

  • Updated Decapitation Head Format.
    • Had to do this to add new features to the Heads themselves.
    • It still works with heads from the old format though!
  • New heads dropped no longer plays a sound.
    • Mainly so these heads can stack together again without issue.
  • Lowered heads' drop chance for "Player Kill" condition.
    • This is to make getting these heads feel more like a goal to work towards for this setting.
    • New drop rates available in Player Kill Drop Chances text file.
  • Updated Piglin Head to use the Vanilla one instead.
    • The old Piglin Head is now Old Piglin Head instead.
  • Changed the Description for the "Not Behind" achievement.
  • Cleaned up outdated lines in "Installing Decapitation" text file.
