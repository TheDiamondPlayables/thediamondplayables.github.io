Changes:

This small update implements the new Admin Book Updates, and improves BrushCover's mechanics. Enjoy!

  • Internal book code now runs on a much better BookMaker addon.
  • Settings are now properly saved as updates are introduced!
  • Fixed Unbreakable tag not working on negative Unbreakable values.
    • In vanilla, negative values are still unbreakable, so yeah.
  • Improved execution on MC-260225 patch.
    • Please vote for the issue to get it patched soon!
  • Cover Limits is now off by default.
    • This won't change the value if you have a previous version BrushCover activated though.
  • Edit Buttons now refresh the book to update modified values as the button is used.
  • Last Modified is now Last Refresh as that's more accurate.
  • Added a gap between the edit command's command and the command response to put those 2 apart.
