Important Note:

  • SpawnerCarts made before this patch can no longer be dropped off due to the new changes.
    • To prevent this, please make sure all SpawnerCarts have been dropped off before updating.
    • Sorry for any inconvenience this might cause you, this won't happen again.

Additions:

  • SpawnerCarts are now properly labeled to avoid clashing with other projects.
    • As a side-effect though, unlabeled SpawnerCarts can't be interacted with anymore.
  • Spawners now pop out if the SpawnerCart is destroyed by any means.
    • This was changed to be consistent with how other entities are popped out of their vehicles in vanilla.

Changes:

  • Cagey Dropoff now gives you more Experience upon completion.
    • You're more likely to miss this one now that the new mechanic exists.
    • So I figured I'd add a bit more experience for those who did notice it.
    • Also, I've reset this achievement's progression.
  • Updated SpawnerCart's particle effects upon caught/dropoff.
  • Optimized some places throughout SpawnerCart's code.
  • SpawnerCarts now only drops the Minecart if doEntityDrops gamerule is activated.

PS: Thanks to @sirfluffypuffs in my Discord Server for suggesting the first iteration of the new spawner dropoff mechanic. [View original suggestion message]
