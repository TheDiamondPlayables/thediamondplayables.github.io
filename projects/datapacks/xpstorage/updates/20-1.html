This update officially brings XPStorage to Minecraft 1.20!
Got some small detail changes for this one, plus some performance improvements, so have fun!

Changes:

  • Drinking an EXP Jar now has extra effects on the drinker.
    • Just some extra details for this so it doesn't feel like drinking a regular Potion.
  • Fixed a bug where XPStorage would repeat past choices before actually reverting settings to default.
    • It should tell you the default settings information instead, since that's now relevant.
  • Fixed a bug that caused the EXP Jar to overlap if an EXP Jar is used to make another EXP Jar.
    • Found it while I was going through the code for the 1.20 update, so I fixed it.
    • It was an embarrasing typo to be honest. I was so confident with that too, I didn't even test it.
  • Improved XPStorage' code with 1.20 stuff.
  • Updated Advancement Background to match 1.20.
