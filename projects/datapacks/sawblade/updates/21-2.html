I try to take care of all edge-cases in vanilla so you can have all the fun, even if you're in some wacky world with a wacky setting, and I actually missed a new one 1.21 (1.20.5 actually, but still). There are so much more of these since 1.20.5 - this is bound to happen at some point.

Changes:

  • Fixed a critical bug causing Sawblade Damage to not run properly if spawnChunkRadius is off. To fix this:
    1. Run: /gamerule spawnChunkRadius 1 for a second
    2. Then, run: /gamerule spawnChunkRadius 0 to turn it off again.
    3. That should fix everything.
